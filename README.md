## 研究目的
探究MapReduce中Combiner对作业性能的影响。

## 研究内容
分析Combiner在MapReduce执行过程中的作用与效果。

具体包括：Combiner是否能 够有效减少Shuffle阶段的数据量？

在不同的key分布（均匀分布与数据倾斜）下，其性能提升效果有何差异？

是否所有场景都适合使用Combiner？
